# Zarahemla Store API
An application that contains the APIs for an e-commerce store called Zarahemla Store.

## Installation
```npm install```

## Running the application
```npm run start``` or  ```npm start```

## API Documentation
The API documentation is generated by `swagger`. You can update the documentation by `npm run swagger` or `node swagger.js`.

## Packages
* express
* body-parser
* swagger-autogen
* swagger-ui-express
* mongodb
* cors
* dotenv
